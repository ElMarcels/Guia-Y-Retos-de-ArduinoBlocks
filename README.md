# Guia Y Retos de ArduinoBlocks
*Creado Por ElMarcels*
-------
### ¿De que trata este repositorio?
➡️ En este repositorio encontraras una pequeña guía de ArduinoBlocks (SteamMakerBlocks) y una serie de retos repartidos en diferentes niveles, para que podais practicar.

## CONTENIDO DEL REPOSITORIO
- Guía ArduinoBlocks
- Retos
  - Nivel 1 (**Aprendemos a usar las cosas basicas**)
    - Reto 1: LEDS
    - Reto 2: LED RGB
    - Reto 3: Zumbador
    - Reto 4: Nivel de Luz
    - Reto 5: Temperatura
    - Reto 6: Nivel de Sonido
    - Reto 7: Potenciometro

  - Nivel 2 (**Combinamos Cosas en un Solo Proyecto**)
    - Reto 1: Nivel de Luz y Temperatura
    - Reto 2: Nivel de Sonido y Luz
    - Reto 3: Potenciometro y Zumbador
    - Reto 4: Potenciometro, LEDS y LED RGB

  - Nivel 3 (**Simulamos la Automatización**)
    - Reto 1
    - Reto 2
    - Reto 3
    - Reto 4
    - Reto 5
  - **PROXIMAMENTE MÁS CONTENIDO**
